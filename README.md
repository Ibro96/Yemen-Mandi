# Yemen-Mandi — To‑Do App

A small, client-side to‑do list application that persists tasks in the browser using localStorage. This repo contains a lightweight HTML/CSS/JavaScript app you can open locally or serve from any static file host.

## Features

- Add new to-dos
- Edit to-dos (double-click or Edit button)
- Toggle complete / active
- Delete to-dos
- Filter (All / Active / Completed)
- Clear completed or clear all
- Persists using localStorage (key: `yemen-mandi-todos-v1`)

## Files

- `index.html` — main app page
- `css/styles.css` — styles
- `js/app.js` — application logic and localStorage handling

## Run locally

1. Clone the repo:
   git clone https://github.com/Ibro96/Yemen-Mandi.git
2. Open `index.html` in your browser, or serve with a static server:
   - Python 3: `python -m http.server 8000`
   - Node: `npx http-server`
3. Visit `http://localhost:8000` (or open the file directly) and start adding tasks.
